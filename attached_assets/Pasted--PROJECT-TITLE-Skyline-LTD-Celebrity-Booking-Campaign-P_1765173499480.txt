# üí∞ **WALLET SYSTEM (USD ONLY)**

### **Users do not hold crypto.

User wallet balance = USD only.**

### Deposit Flow:

1. User enters USD amount
2. Selects coin: **BTC, ETH, USDT(TRC20)**
3. System fetches live crypto price
4. Show exact crypto amount required
5. Show admin‚Äôs crypto wallet address
6. User sends crypto
7. User clicks **Confirm Payment**
8. System creates **Pending Deposit Task**
9. Admin manually verifies incoming payment
10. If approved ‚Üí wallet credited in USD
11. Auto-generate PDF receipt

### Admin controls:

* BTC wallet address
* ETH wallet address
* USDT wallet address

Editable inside admin panel.
